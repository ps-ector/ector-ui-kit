# Tailwind configuration ( TODO )

If you want to use Ector UI Kit in a native way you have to setup this configuration in your project.

## tailwind.config.js

```js
module.exports = {
  purge: {
    content: [
      // your path files
    ],
  },
  theme: {
    extend: {
      colors: {
        primary: "#47D7AC",
        secondary: "#11111F",
      },
      fontFamily: {
        sans: ["Roboto", "sans-serif"],
      },
    },
  },
};
```
